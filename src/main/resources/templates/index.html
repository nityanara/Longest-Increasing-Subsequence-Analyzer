💡 What is LIS?

The Longest Increasing Subsequence (LIS) is a core problem in Dynamic Programming.

It finds the longest subsequence in a sequence such that all elements are strictly increasing.

Example:
Input → 3 10 2 1 20
LIS → 3 10 20

⚙️ Logic Used

This project uses Java-based logic with:

  • 📌 Dynamic Programming (O(n²))
  • ⚡ Optimized Binary Search (O(n log n))

The algorithm works by tracking increasing sequences and updating the best possible subsequence at each step.
